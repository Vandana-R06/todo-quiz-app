🚀 To-Do + Quiz Web App

📌 Project Overview

This project is a multi-functional web application that combines a To-Do list and an interactive Quiz system in a single interface. It is designed to improve productivity while also testing user knowledge in a fun and engaging way.

---

✨ Features

📝 To-Do List

- Add new tasks
- Mark tasks as completed ✔️
- Delete individual tasks ❌
- Clear all tasks 🧹
- Persistent storage using LocalStorage 💾 (tasks remain after refresh)

❓ Quiz System

- Multiple-choice questions (MCQs)
- Timer-based quiz ⏳
- Auto-submit when time ends
- Score calculation 🎯
- Interactive UI

🎨 UI/UX Enhancements

- Dark Mode toggle 🌙
- Responsive and clean design
- Progress bar for quiz 📊

---

🛠️ Technologies Used

- HTML5 – Structure of the web page
- CSS3 – Styling and layout
- JavaScript (Vanilla JS) – Logic and interactivity
- LocalStorage API – Data persistence

---

▶️ How to Run the Project

1. Clone or download the repository
2. Open the project folder
3. Double-click on "index.html" or open it in any browser
